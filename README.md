# Installations
```sh
# Install everything
curl -s https://raw.githubusercontent.com/LeonardsonCC/ansible-playbooks/main/install.sh | bash -s -- --all

# Install only neovim
curl -s https://raw.githubusercontent.com/LeonardsonCC/ansible-playbooks/main/install.sh | bash -s -- --neovim
```

# Requirements
- Ansible
